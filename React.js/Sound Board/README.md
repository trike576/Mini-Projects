# Sound Board

A React.js soundboard application that allows users to play sounds by clicking colorful sound pads or by pressing the corresponding keyboard keys.

## Features

* 14 unique sound pads
* Keyboard support
* Colorful and responsive UI
* Audio playback using JavaScript Audio API
* Interactive button animations

## Technologies Used

* React.js
* JavaScript
* CSS3
* Vite

## Concepts Practiced

* React Components
* Props
* Rendering Lists with map()
* Event Handling
* Keyboard Events
* Working with Media Assets
* Inline Styling

## Installation

```bash
npm install
npm run dev
```

## Project Goal

This project was built to practice React fundamentals such as component rendering, event handling, working with assets, and user interaction.
